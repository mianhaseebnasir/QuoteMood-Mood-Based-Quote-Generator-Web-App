# 📌 Project Overview

QuoteMood is a simple and user-friendly web application designed to generate quotes based on the user’s mood. It helps users feel motivated, calm, inspired, or happy by displaying relevant quotes in a clean UI.

This project is ideal for beginners learning JavaScript DOM manipulation and UI interaction.

# 🎯 Features
- 🎭 Mood-based quote filtering (Happy, Motivated, Calm, Inspired)
- 🔄 Generate new quotes instantly
- ❤️ Mark quotes as favorite
- 📋 Copy quote to clipboard
- ✨ Clean and modern UI design
- ⌨️ Spacebar shortcut for new quotes

# 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Google Fonts (Inter)
